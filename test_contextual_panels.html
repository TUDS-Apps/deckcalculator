üìã Panel States & Content

State 1: Drawing Mode

Trigger: !appState.isShapeClosed
Shows: Basic specs (height, footing, attachment), Drawing instructions
Hides: Advanced options, Generate Plan button, Stair configuration
Purpose: Focus on deck outline creation with minimal distractions

State 2: Wall Selection Mode

Trigger: Shape closed && selectedWallIndex === -1
Shows: Clear wall selection instructions with helpful tips
Hides: All specification details, Advanced options
Purpose: Guide user to select attachment wall

State 3: Specification Mode

Trigger: Shape closed && wall selected && !structuralComponents
Shows: All deck specifications, Large "Generate Plan" button, Collapsible advanced options
Hides: Stair configuration
Purpose: Comprehensive specification setting before plan generation

State 4: Plan Generated Mode

Trigger: structuralComponents && !error
Shows: Project summary, "Add Stairs" button, "Modify Specifications" option
Features: Collapsible spec editor for modifications
Purpose: Highlight completed plan and next action options

State 5: Stair Configuration Mode

Trigger: stairPlacementMode === true
Shows: Stair configuration panel, Placement instructions, Cancel button
Hides: Most deck specifications, BOM details
Purpose: Focus exclusively on stair configuration
